Краткое описание ТГ-бота:

Что это:
Финансовый бот для учета личных доходов и расходов в Telegram.
ОСНОВНЫЕ ФУНКЦИИ:

1. Учет операций:
   - Добавление доходов
   - Добавление расходов

2. Статистика:
   - Просмотр баланса
   - Круговые диаграммы доходов/расходов
   - Детальная статистика по категориям

3. Категории:
   - Еда, Транспорт, Жилье
   - Зарплата, Инвестиции, Подарки
   - И другие категории

как работает:
1. Пользователь отправляет /start
2. Появляется меню с кнопками
3. Выбирает "Доход" или "Расход"
4. Выбирает категорию
5. Вводит сумму
6. Бот сохраняет данные и показывает статистику

хранение данных:
- Все данные хранятся в data.json
- Каждый пользователь имеет свою историю операций
- Автоматическое сохранение после каждой операции

цель:
Помочь пользователям легко отслеживать свои финансы прямо в Telegram без сложных приложений.

Мой проект это - упрощенный домашний бухгалтер в телеграме с графиками
